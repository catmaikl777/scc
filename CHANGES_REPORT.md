## 📱 Как использовать новые функции

### Приватные комнаты:
1. Нажмите кнопку 🔒 в панели эмодзи
2. Выберите комнату из списка или создайте новую
3. Введите пароль если требуется

### Система друзей:
1. Нажмите кнопку 👥 в панели эмодзи
2. Просмотрите список друзей и их статус
3. Нажмите 💬 для быстрого написания сообщения онлайн другу

### Показ экрана:
1. Присоединитесь к видеозвонку
2. Нажмите кнопку 📱 в панели эмодзи
3. Выберите окно или экран для демонстрации
4. Другие участники увидят ваш экран в отдельном окне
