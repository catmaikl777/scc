# 🤖 AI Интеграция в Космический Кот Чат

## ✅ Что добавлено

### 🧠 AI Помощник
- **Groq API** (рекомендуется) - 6000 запросов/час бесплатно
- **Google Gemini API** - 15 запросов/минуту бесплатно  
- **Hugging Face API** - 1000 запросов/месяц бесплатно
- Fallback система для надежности

### 🌍 Переводчик
- **MyMemory API** - бесплатный переводчик
- Поддержка основных языков (EN, ES, FR, DE, ZH)
- Автоопределение исходного языка

### ☁️ Погода
- **OpenWeatherMap API** - 1000 запросов/день бесплатно
- Поддержка геолокации и поиска по городу
- Подробная информация о погоде

## 🚀 Быстрая настройка

### 1. Создайте файл .env
```bash
# Скопируйте .env.example в .env
cp .env.example .env
```

### 2. Получите API ключи

#### Groq API (рекомендуется)
1. Перейдите на https://console.groq.com/
2. Зарегистрируйтесь бесплатно
3. Создайте API ключ
4. Добавьте в .env: `GROQ_API_KEY=gsk_your_key_here`

#### Google Gemini API
1. Перейдите на https://makersuite.google.com/app/apikey
2. Создайте API ключ
3. Добавьте в .env: `GEMINI_API_KEY=your_key_here`

#### OpenWeatherMap API
1. Перейдите на https://openweathermap.org/api
2. Зарегистрируйтесь бесплатно
3. Получите API ключ
4. Добавьте в .env: `OPENWEATHER_API_KEY=your_key_here`

### 3. Запустите сервер
```bash
npm start
```

## 🎯 Как использовать

### AI Помощник
1. Нажмите кнопку 🤖 в панели эмодзи
2. Введите вопрос
3. AI ответит в общем чате

### Переводчик
1. Нажмите кнопку 🌍 в панели эмодзи
2. Введите текст для перевода
3. Выберите целевой язык
4. Результат появится в чате

### Погода
1. Нажмите кнопку ☁️ в панели эмодзи
2. Разрешите геолокацию или введите город
3. Информация о погоде появится в чате

## 🐛 Устранение неполадок

### AI не отвечает
1. Проверьте API ключи в .env
2. Убедитесь что сервер перезапущен
3. Проверьте лимиты API

### Переводчик не работает
- MyMemory API бесплатный, но может быть медленным
- Попробуйте позже если превышен лимит

### Погода не загружается
1. Проверьте OPENWEATHER_API_KEY
2. Разрешите геолокацию в браузере
3. Попробуйте ввести город вручную
