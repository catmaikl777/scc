# 🆓 Настройка бесплатных API

## 🤖 AI Помощник (Бесплатные варианты)

### 1. Groq API (Рекомендуется) 
- **Лимит**: 6000 запросов/час бесплатно
- **Регистрация**: https://console.groq.com/
- **Получение ключа**: API Keys → Create API Key
- **Переменная**: `GROQ_API_KEY=gsk_your_key_here`

### 2. Google AI Studio (Gemini)
- **Лимит**: 15 запросов/минуту бесплатно
- **Регистрация**: https://aistudio.google.com/
- **Получение ключа**: Get API Key
- **Переменная**: `GEMINI_API_KEY=your_key_here`

### 3. Hugging Face (Fallback)
- **Лимит**: 1000 запросов/месяц бесплатно
- **Регистрация**: https://huggingface.co/
- **Получение ключа**: Settings → Access Tokens
- **Переменная**: `HUGGINGFACE_API_KEY=hf_your_key_here`

## 🌍 Переводчик

### MyMemory API (Полностью бесплатный)
- **Лимит**: 1000 символов/день без регистрации
- **Лимит**: 10000 символов/день с email
- **Регистрация**: Не требуется (или https://mymemory.translated.net/)
- **Переменная**: Не требуется

## ☁️ Погода

### OpenWeatherMap API
- **Лимит**: 1000 запросов/день бесплатно
- **Регистрация**: https://openweathermap.org/api
- **Получение ключа**: API Keys
- **Переменная**: `OPENWEATHER_API_KEY=your_key_here`

## 📝 Настройка .env файла

Создайте файл `.env` в корне проекта:

```bash
# База данных (обязательно)
DATABASE_URL=postgresql://user:password@localhost:5432/chatdb

# AI API (выберите один или несколько)
GROQ_API_KEY=gsk_your_groq_key_here
GEMINI_API_KEY=your_gemini_key_here
HUGGINGFACE_API_KEY=hf_your_huggingface_key_here

# Погода (опционально)
OPENWEATHER_API_KEY=your_openweather_key_here

# Порт сервера (опционально)
PORT=3000
```

## 🚀 Быстрая настройка (только Groq)

1. Зарегистрируйтесь на https://console.groq.com/
2. Создайте API ключ
3. Добавьте в `.env`:
   ```bash
   GROQ_API_KEY=gsk_your_key_here
   ```
4. Перезапустите сервер: `npm start`

## 🔄 Как работает система

1. **AI запросы** обрабатываются в порядке приоритета:
   - Groq API (быстрый, хороший лимит)
   - Gemini API (качественный, средний лимит)
   - Hugging Face API (fallback)
   - Локальные ответы (если все API недоступны)

2. **Переводчик** использует бесплатный MyMemory API

3. **Погода** использует OpenWeatherMap или показывает заглушку

## ⚡ Без настройки API

Даже без API ключей чат будет работать:
- AI будет давать базовые ответы на основе правил
- Переводчик покажет заглушку
- Погода покажет случайные данные

## 📊 Мониторинг лимитов

- **Groq**: https://console.groq.com/settings/limits
- **Gemini**: https://aistudio.google.com/app/apikey
- **OpenWeather**: https://openweathermap.org/api/statistics

## 🔒 Безопасность

- Никогда не публикуйте API ключи в коде
- Используйте `.env` файл (он в `.gitignore`)
- Регулярно обновляйте ключи
- Мониторьте использование API

## 🆘 Проблемы

### AI не отвечает
1. Проверьте правильность API ключа
2. Проверьте лимиты на сайте провайдера
3. Посмотрите логи сервера

### Ошибки API
- Система автоматически переключается на fallback
- Проверьте интернет соединение
- Убедитесь что ключи активны

## 💡 Советы

1. **Groq** - лучший выбор для начала (быстрый и щедрый лимит)
2. **Gemini** - хорошее качество ответов
3. Настройте несколько API для надежности
4. Мониторьте использование лимитов
