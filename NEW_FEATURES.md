## 🚀 Как использовать

### Приватные комнаты:
1. Нажмите 🔒 в панели эмодзи
2. Введите название комнаты
3. Установите пароль (опционально)
4. Поделитесь ID комнаты с друзьями

### Система друзей:
1. Нажмите 👥 в панели эмодзи
2. Выберите пользователя из списка
3. Отправьте запрос в друзья
4. Ждите подтверждения

### Турниры:
1. Нажмите 🏆 в панели эмодзи
2. Выберите тип игры
3. Установите количество участников
4. Дождитесь набора игроков

### Видео-сообщения:
1. Нажмите 📹 в панели эмодзи
2. Разрешите доступ к камере
3. Нажмите "Начать запись"
4. Остановите запись и отправьте
