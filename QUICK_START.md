# ⚡ Быстрый запуск

## 1. Установка зависимостей
```bash
npm install
```

## 2. Настройка базы данных
```bash
# Создайте PostgreSQL базу данных
createdb chatdb

# Установите переменную окружения
export DATABASE_URL="postgresql://user:password@localhost:5432/chatdb"
```

## 3. Запуск сервера
```bash
npm start
```

Чат будет доступен по адресу: http://localhost:3000

## 4. Настройка AI (опционально)

### Быстрая настройка с Groq (рекомендуется)
1. Зарегистрируйтесь на https://console.groq.com/
2. Создайте API ключ
3. Создайте файл `.env`:
   ```bash
   DATABASE_URL=postgresql://user:password@localhost:5432/chatdb
   GROQ_API_KEY=gsk_your_key_here
   ```
4. Перезапустите сервер: `npm start`

### Тестирование API
```bash
npm run test-apis
```

## 5. Возможности без API

Даже без настройки API чат полностью функционален:
- ✅ Текстовые сообщения
- ✅ Голосовые сообщения  
- ✅ Видеозвонки
- ✅ Загрузка файлов
- ✅ Игры и опросы
- ✅ Стикеры
- ⚠️ AI даст базовые ответы
- ⚠️ Переводчик покажет заглушку
- ⚠️ Погода покажет случайные данные

## 🆘 Проблемы?

1. **Ошибка подключения к БД**: Проверьте DATABASE_URL
2. **AI не отвечает**: Проверьте API ключи в .env
3. **Порт занят**: Измените PORT в .env или используйте другой порт

## 🎯 Минимальная настройка

Для работы нужен только PostgreSQL:
```bash
# .env
DATABASE_URL=postgresql://user:password@localhost:5432/chatdb
```

Все остальное опционально!
