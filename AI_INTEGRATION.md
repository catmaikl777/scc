# 🤖 AI Интеграция

## Настройка OpenAI API

### 1. Получение API ключа
1. Зарегистрируйтесь на https://platform.openai.com
2. Создайте новый API ключ в разделе API Keys
3. Скопируйте ключ

### 2. Настройка в проекте
Добавьте в файл `.env`:
```bash
OPENAI_API_KEY=sk-your-openai-api-key-here
```

### 3. Возможности AI помощника
- ✅ Ответы на любые вопросы
- ✅ Помощь с программированием
- ✅ Объяснение сложных тем
- ✅ Генерация идей
- ✅ Перевод текста
- ✅ Решение задач

### 4. Использование
1. Нажмите кнопку 🤖 в панели эмодзи
2. Введите ваш вопрос
3. Получите ответ от AI

## Дополнительные API

### Google Translate API
```bash
GOOGLE_TRANSLATE_API_KEY=your-google-translate-key
```

### OpenWeatherMap API
```bash
OPENWEATHER_API_KEY=your-openweather-key
```

## Fallback режим
Если API ключи не настроены, система работает с базовыми ответами.

## Безопасность
- Никогда не публикуйте API ключи в коде
- Используйте переменные окружения
- Регулярно обновляйте ключи
