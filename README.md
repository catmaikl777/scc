# 🚀 Космический Кот Чат

Современный веб-чат с видеозвонками и системой уведомлений внутри приложения.

## ✨ Возможности

### 💬 Чат
- Мгновенные сообщения в реальном времени
- Голосовые сообщения
- Отправка файлов (изображения, видео, документы)
- Эмодзи и стикеры
- Приватные сообщения

### 📞 Видеозвонки
- Групповые видеозвонки
- Индивидуальные звонки
- Переключение камер
- Управление аудио/видео
- WebRTC соединение

### 🎮 Игры и развлечения
- Крестики-нолики
- Цепочка слов
- Викторины
- Загадки
- Опросы и голосования

### 🔔 Уведомления
- Звуковые уведомления
- Браузерные уведомления
- Уведомления внутри чата
- Настройки звука

### 🎨 Интерфейс
- Космическая тема с неоновыми эффектами
- Адаптивный дизайн
- Анимации и частицы
- Темная тема

## 🛠️ Технологии

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Backend**: Node.js, WebSocket (ws)
- **База данных**: PostgreSQL
- **WebRTC**: Для видеозвонков
- **PWA**: Service Worker, Web App Manifest

## 🚀 Быстрый старт

### Установка
```bash
git clone <repository-url>
cd space-cat-chat
npm install
```

### Настройка базы данных
```bash
# Создайте PostgreSQL базу данных
# Установите переменную окружения
export DATABASE_URL="postgresql://user:password@localhost:5432/chatdb"
```

### Запуск
```bash
npm start
```

Приложение будет доступно по адресу `http://localhost:3000`

## 🔧 Локальные утилиты

Приложение включает мощные локальные утилиты:

- **Локальный кэш** - быстрое хранение данных в памяти
- **Автобэкапы** - ежедневное сохранение данных чата
- **Статистика** - анализ активности и производительности
- **Логирование** - отслеживание событий и ошибок
- **Утилиты файлов** - валидация и обработка файлов
- **Безопасность** - защита от спама и вредоносного контента
- **Аналитика** - отслеживание действий пользователей
- **Планировщик** - автоматическое выполнение задач
- **Отчеты** - автоматическая генерация отчетов
- **Мониторинг** - отслеживание производительности
- **Уведомления** - система внутренних уведомлений
- **IP утилиты** - работа с IP адресами и блокировка

### API эндпоинты:
- `GET /api/stats/daily` - статистика за день
- `GET /api/stats/weekly` - статистика за неделю
- `POST /api/backup/create` - создать бэкап
- `GET /api/backup/list` - список бэкапов
- `GET /api/cache/stats` - статистика кэша
- `GET /api/logs` - получить логи
- `GET /api/performance` - метрики производительности
- `GET /api/analytics` - аналитика пользователей
- `GET /api/security` - информация о безопасности
- `GET /api/tasks` - список запланированных задач
- `GET /api/reports/daily` - ежедневный отчет
- `GET /api/reports/weekly` - недельный отчет
- `GET /api/reports/health` - отчет о состоянии системы
- `GET /api/notifications` - история уведомлений
- `POST /api/notifications/{id}/read` - отметить как прочитанное

## 🎮 Игры

### Доступные игры:
- **Крестики-нолики** - классическая игра для двух игроков
- **Цепочка слов** - составление слов по последней букве
- **Викторина** - вопросы с вариантами ответов
- **Загадки** - отгадывание загадок с подсказками

### Опросы:
- Создание опросов с несколькими вариантами
- Голосование в реальном времени
- Ограничение по времени

## 📞 Видеозвонки

### Возможности:
- Групповые звонки до 16 участников
- Индивидуальные звонки
- Переключение между камерами
- Отключение/включение микрофона и камеры
- Адаптивная сетка видео

### Требования:
- Современный браузер с поддержкой WebRTC
- Разрешение на доступ к камере и микрофону
- Стабильное интернет-соединение

## 🔧 Конфигурация

### Переменные окружения:
```bash
DATABASE_URL=postgresql://user:password@localhost:5432/chatdb
PORT=3000
NODE_ENV=production
```

### Настройки WebRTC:
Используются публичные STUN/TURN серверы для обеспечения соединения через NAT.
